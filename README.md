# X Cup Markets Frontend

Static frontend prototype for a World Cup-themed prediction market on X Layer.

## Files

- `index.html` contains the page structure and placeholder market data.
- `styles.css` contains the full responsive layout and X Layer-inspired theme.
- `app.js` contains small prototype interactions for tabs and the trade slip.

## Backend handoff notes

- Replace `.image-placeholder` blocks with real campaign, match, NFT, and quest images.
- Replace hardcoded market rows with API-driven market data.
